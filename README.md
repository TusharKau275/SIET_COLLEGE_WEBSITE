# 🎓 College Website Project

### 🏆 WebCraft Hackathon 2026

---

## 📌 Project Overview

This project is a modern, responsive, and feature-rich **College Website** developed by our team during the **WebCraft Hackathon 2026**.

The website is designed to enhance digital interaction between students, faculty, and visitors by providing real-time updates, smooth navigation, and an engaging user interface.

---

## 🚀 Key Features

### 🎥 Dynamic Hero Section

* Interactive **auto-playing slider/carousel**
* Showcases:

  * Tech Fests
  * Placement Highlights
  * Campus Tour
* Includes **manual controls** for user navigation
* Smooth transitions and animations

---

### 🔔 Live Alerts / Notice Popup

* Displays important announcements on page load
* Implemented using:

  * Toast notifications
  * Flashcard popups / scrolling ticker
* Examples:

  * Exam schedule updates
  * Holiday notices
  * Club events
* Fully **dismissible by user**

---

### 📱 Responsive Navbar with Hamburger Menu

* **Desktop View:**

  * Clear navigation links:

    * Academics
    * Admissions
    * Life @ Campus
    * Login

* **Mobile View:**

  * Collapsible **Hamburger Menu**
  * Smooth slide-in / dropdown animation

* Fully responsive across all devices

---

### 🌙 Dark / Light Mode Toggle

* One-click **theme switcher**
* Seamless transition between:

  * Light Mode ☀️
  * Dark Mode 🌙
* Applies across:

  * Text
  * Backgrounds
  * Cards & UI components

---

### 🔗 API / Data Integration

* Integrated external data using public APIs
* Examples of implementation:

  * 🌦️ Weather API → Shows real-time campus weather
  * 📰 News API → Displays latest tech/news updates
  * 📊 JSON Placeholder → Student directory / notice board
* Demonstrates real-world **data fetching and rendering**

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Styling:** CSS3 (Flexbox, Grid, Animations)
* **APIs:** REST APIs (Weather / News / JSON Placeholder)
* **Version Control:** Git & GitHub
